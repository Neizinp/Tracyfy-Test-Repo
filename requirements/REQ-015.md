# Suspect Link Dependency Highlighting

## Description
Visual cues for potential verification gaps.

## Requirement Text
The system shall visually flag established links as "suspect" whenever the source artifact of the link is modified.

## Rationale
Directly informs the user of which verification items need re-run.
