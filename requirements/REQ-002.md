# Artifact-to-Path Mapping

## Description
Translating Unique IDs to structured filesystem paths.

## Requirement Text
The system shall map each artifact ID to a dedicated filesystem path using the pattern: `/{type}/{id}.md`.

## Rationale
Predictable mapping is required for direct file access and indexing.
