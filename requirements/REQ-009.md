# Typed Relationship Support

## Description
Categorizing links with specific relationship types.

## Requirement Text
The system shall support the assignment of relationship types (e.g., "satisfies", "verifies") to every traceability link.

## Rationale
Adds semantic meaning to the links for better analysis.
