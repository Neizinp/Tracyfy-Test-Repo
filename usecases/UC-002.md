# Establish Traceability Link

## Description
Connecting two artifacts to create a relationship.

## Actor
System Architect

## Preconditions
At least two artifacts exist in the project.

## Main Flow
1. User navigates to the 'Links' page.
2. User selects a 'Source' artifact.
3. User selects a 'Target' artifact.
4. User chooses the link type (e.g., 'verifies').
5. User submits the link creation.

## Alternative Flows
Attempting to link an artifact to itself: System displays error.

## Postconditions
A new entry is added to the links registry.
