# Export Verification Matrix

## Description
Generating a compliance document for auditing.

## Actor
Quality Assurance Lead

## Preconditions
Requirements have linked Test Cases with results.

## Main Flow
1. User opens the 'Export' modal.
2. User selects 'Excel' format.
3. User toggles 'Include Verification Matrix'.
4. User clicks 'Download'.
5. System generates and triggers file transfer.

## Alternative Flows
Large project: System shows progress bar during generation.

## Postconditions
An Excel file is downloaded to the local machine.
